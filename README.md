## Installation
> All the `code` required to get started

### Clone

> Clone this repo to your local machine using `https://github.com/Kubanacode/quantumdemi.git`

### Setup

> Install all package first

```shell
$ npm install
OR
$ yarn add
```
> Run

```shell
$ npm start
OR
$ yarn start
```
